肌診断アプリ（SkinAI）

個人開発している美容 × AI アプリです。  
React Native（Expo）と Firebase を使用し、  
肌の写真からAI風の肌タイプ分析、スコア診断、履歴保存ができる仕組みを実装しています。

---

　使用技術

- React Native (Expo)
- TypeScript
- Zustand（状態管理）
- Firebase Authentication
- Firebase Firestore
- Expo Image Picker
- Expo Router

---

　主な機能

- 肌写真のアップロード
- 簡易AI分析（肌タイプ・乾燥・油分・赤み）
- 質問によるスコア診断
- 診断結果を Firestore に保存
- 履歴一覧画面
- 履歴詳細画面

---

 画面構成

- ホーム画面（診断開始／履歴）
- 写真アップロード画面
- 診断結果画面
- 質問画面
- 履歴一覧画面
- 履歴詳細画面

---

 目的
「美容で人に希望を与えたい」という思いから開発したアプリ。  
肌の悩みを可視化し、改善のきっかけをつくることを目指しています。
